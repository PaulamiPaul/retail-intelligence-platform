# Retail Intelligence Platform

An AI-powered analytics project built on the Olist Brazilian E-commerce dataset.

## Project Overview
End-to-end retail analytics platform covering data exploration, automated pipelines, 
AI agents, and an interactive business dashboard.

## Dataset
Olist Brazilian E-commerce Public Dataset — 99,441 orders across 8 relational tables.
Source: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

## Tech Stack
Python, pandas, matplotlib, DuckDB, LangChain, Streamlit

## Phases
- Phase 1 ✅ Data exploration & EDA
- Phase 2 🔄 Data pipeline & cleaning
- Phase 3 ⏳ AI agents & automations
- Phase 4 ⏳ Dashboard & visualisation
